# CodeLogoProject
Coding Club Logo designed using just Python code.

## Preview
![Coding Club Logo](codelogo.png)

## Technologies Used
- Python 3.9
- matplotlib
- numpy
- math

## How to Run
```bash
pip install matplotlib numpy
python3 codelogo.py
```
The logo will be exported as:
```
codelogo.png
```

## Output
- Logo as PNG
- Fully code-generated
